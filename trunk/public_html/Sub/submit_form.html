5면 이내의 원고분량

2칼럼/면

사용 워드프로세서 : MSWord, HWP(Ver.97) 작성후 pdf로 변환하여 제출요
(PDF파일의 수식이나 그림이 원본과 비교하여 이상이 없는 것을 반드시 확인바람)

MS Word 사용 : 국문은 바탕체, 영문은 Times New Roman

HWP 사용시 : 국문과 영문은 모두 신명조체

 세부 사항
참고용 Template : MS Word양식, HWP양식

국문 저자명 및 소속기관명 : 글자 크기 10, 가운데 정렬

영문 제목 : 글자 크기 14, 가운데 정렬

영문 저자명 및 소속기관명 : 글자 크기 10, 가운데 정렬
